UltraEd — Nintendo 64 Game Engine & Level Editor

A from-scratch level editor and game engine targeting real Nintendo 64 hardware. Scene editing, automatic asset tracking, and a C scripting API with actor lifecycle hooks ($Start, $Update, $Input, $Collide). 2006★ Original paper included

S.A.D — A GPS Speed Awareness Device

A homebrew device that tells you the speed limit of the road you're on — built years before cars did this themselves. A floor-mounted base unit reads its position from a GPS module, parses the raw NMEA 0183 stream, and matches your coordinates against street “boundary boxes” stored in an I²C EEPROM, then beams the speed limit and street name to a dashboard LCD over a single phone cable.

The base runs four different Atmel 8051 microcontrollers — one parsing GPS, one matching the map, one handling EEPROM map updates over a DB9 serial link from a custom PC tool, and one driving the remote display. Each chip model (AT89C2051/51/52/55) was picked to fit its code module. All of it hand-wired on breadboards and stuffed into a project box.

  • 8051 / assembly
  • GPS / NMEA
